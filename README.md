# BonusCarnisApp || Beefslider - Barcode-Based Invoicing App for Meat Products

Welcome to the Beefslider README document. This file provides a comprehensive overview of the Beefslider application, its objectives, functionality, operating environment and essential information.

## Objectives

The primary objectives of Beefslider were as follows:

- Develop a high-quality Android application that enables accurate invoicing of meat products using barcode scanning.
- Provide an intuitive and aesthetically pleasing user interface that offers a seamless user experience.
- Gather essential data from barcodes, including product name, weight, and price.
- Establish an efficient storage mechanism for transforming barcode data into processed files.
- Allow users to share product information via email.
- Facilitate flexible and straightforward management of the EAN database.
- Editing EAN codes (name, weight, price) and modifying saved files.

## Functionality

Beefslider is an Android application that allows precise invoicing of meat products using barcode scanning. The application functions as follows:

- The user initiates the Beefslider application on their Android device.
- The application employs the device's camera to scan barcodes.
- The scanned barcode is compared to stored product information.
- If the barcode matches, the product's name, weight, and price are recorded into a file, creating a comprehensive meat product list.
- Users can effortlessly share the generated list via email.

## Operating Environment

The operational environment for the Beefslider application consists of the following:
- An Android phone equipped with a camera.
- Internet connection for utilizing email functionality.
- The application functions optimally with at least Android 10 operating system (API level 29).
